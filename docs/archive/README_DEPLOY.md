## 🔧 After Deployment

### Update Wallet Addresses

1. Edit `config/payment_addresses.json`
2. Add your real Exodus wallet addresses:
   - Ethereum: Open Exodus → ETH → Receive
   - Bitcoin: Open Exodus → BTC → Receive
   - etc.
3. Redeploy (automatic on most platforms)

### Test Your API

```bash
# Health check
curl https://your-app-url/health

# Get pricing
curl https://your-app-url/api/payments/pricing

# Create free subscription
curl -X POST https://your-app-url/api/payments/subscribe \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","tier":"free"}'
```
